# leaf
YHD workflow tool, organize projects PRD，Prototype &amp; Visual.
# 目录结构
- window 挂载的CrossShare目录
- projects 对应的项目存储目录

# install Mongodb

http://docs.mongodb.org/manual/administration/install-on-linux/

# 配置使用本地mongodb
|
|- config.json

```
{
	"mongodb" : "mongodb://localhost:27017/leaf"
}
```

# 启动Leaf
```
  ./bin/www
```
# 访问
```
  http://localhost:9001
```
